
# Welcome To Spill
The Spill plug-in is an AAX Native/DSP plug-in that allows users to adjust the individual gain of each channel in a multi-channel track in Pro Tools. From a 2-channel stereo track up to a 16-channel immersive track such as 9.1.6 or 3rd Order Ambisonics.

The Spill plug-in operates with zero latency with very low DSP usage.

See [Supported Track Formats](./usage.md#supported-track-formats) for the complete list of formats.
